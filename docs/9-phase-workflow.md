## Why Multi-Agent | 為什麼要多 Agent

單一 AI 既當作者又當審查者，會有「自己檢查自己」的盲點。本框架把工作拆成**獨立角色**，
讓「寫」與「審」由不同 agent instance 承擔，避免連坐偏誤。
